# Quinto Teste de Automação com Robot Framework e Python
Automação desenvolvida para abrir quatro sites em sequência e interagir com elementos no navegador Google Chrome.

Ferramentas Utilizadas:
- Robot Framework: Framework utilizado para estruturar e executar os testes automatizados.
- Python: Linguagem de programação usada para configurar o ambiente e integrar bibliotecas.
- SeleniumLibrary: Biblioteca do Robot Framework responsável por automatizar ações no navegador.
- ChromeDriver: Componente que permite ao Selenium controlar o navegador Google Chrome.
- Visual Studio Code: Ambiente de desenvolvimento utilizado para escrever e organizar os scripts de automação.
